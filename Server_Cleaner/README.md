# 🧹 Base_Folder Automation Script

This Python script is a directory automation tool that helps in **cleaning directories by identifying and deleting duplicate files** using MD5 checksum comparison. It can be scheduled to run at regular intervals using the `schedule` module.

---

## 📌 Features

- Scans a specified directory recursively
- Calculates checksums for each file
- Detects duplicate files
- Deletes duplicates safely
- Supports automated scheduling
- Generates log files with scan information
